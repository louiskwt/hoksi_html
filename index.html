HTML簡介

HTML 全名HyperText Markup Language） 係目前用喺網頁嘅主流標記語言。HTML元素係網頁嘅基本組成模塊。HTML嘅書寫形式一般由封閉嘅標籤構成，而且標籤會寫喺兩個尖括號裏面，譬如。而內容就會寫喺封閉標籤入面。

HTML標記由幾個關鍵部件構成，主要係元素（以及佢嘅屬性）

HTML元素最普遍嘅形式一般含有三樣嘢：起始標籤 ( < )、結束標籤 (&rt)；寫喺起始標籤裏面嘅某啲屬性, 例如 style = “color: black;”

關於屬性請參考 HTML屬性簡介

元素例子：

                      
                        <h1>我喺一個HTML元素</h1>

h Tag嘅運用

H tag 係用來顯示標題嘅元素

H tag 有 6 隻 h1 - h6

Example Code

                    
                    // Example HTML code


                    <h1>h1喺咁</h1>
                    <h2>h2喺咁</h2>
                    <h3>h3喺咁</h3>
                    <h4>h4喺咁</h4>
                    <h5>h5喺咁</h5>
                    <h6>h6喺咁</h6>

實際效果

  • h1喺咁

  • h2喺咁

  • h3喺咁

  • h4喺咁

  • h5喺咁
  • h6喺咁

實際效果會因程式預設嘅style或者瀏覽器(browser)嘅設定而有所不同

Div Tag嘅運用

Div 係叫 divider 元素，可以用來將唔用元素分隔開

一般來講加 div 會開新一行，但其實 div 的作用唔止係咁多

Example Code

                
                    <div>
                        Question: 1 + 1 =
                        <div>
                            A : 1
                        </div>
                        <div>
                            B : 2
                        </div>
                    </div>

實際效果

Question: 1 + 1 =
    A: 1
    B: 2

^以上效果經過加工

Br 同 Hr嘅運用

其實除咗用 div 去分行，其實可以用 br tag 去開新一行

至於 hr tag 會畀一條分隔線你，當你要分開幾條題目或者唔同section時可能會用到

Example Code

                  
                      <h4>Question 1： 1 ＋ 1 = </h4>

                      <br> // 用br開新一行

                      <h4>選項</h4>

                      <br>

                      <h5> A : 2 </h5>

                      <br>

                      <h5> B : 1 </h5>

                      <hr style="border: 1px solid black;" >  //黑色分隔線

                      <h5> Answer: 2 </h5>

實際效果

Question 1 + 1 =


選項


A : 2

B : 1
Answer : 2

^以上效果經過加工

HTML嘅 style屬性

HTML tag 喺可以通過設定屬性去加入附加信息或者改變某些設定

以學斯嘅使用情況來講，可以改style屬性來排位

用style屬性嘅方式係： style = "特性：數值"

Example Code

                  
                      <h3>冇屬性嘅h3</h3>

                      //style="text-align: center" 將個h3擺去中間

                      <h3 style="text-align: center">被置中嘅h3</h3>

實際效果

冇屬性嘅h3

被置中嘅h3

^以上效果經過加工

用style來排位

喺學斯HTML入面，用 style屬性加入自訂嘅style係比較穩陣同好嘅方法

除咗 style="text-align: center" 之外，仲可以用margin來排位

margin係指html元素另一個元素之間的距離，可以細分成margiin-top, margin-right, margin-bottom, 同margin-left

Example Code

                  
                      <h3 style="margin-bottom: 0px">上下零距嘅h3</h3>
                      <h3 style="margin-top: 0px">上下零距嘅h3h3</h3>

                      <hr>

                      <h3 style="margin-bottom: 50px">分得好開嘅h3</h3>
                      <h3 style="margin-top: 50px">分得好開嘅h3</h3>
                     
                      <hr>

                      <h3>普通嘅h3</h3>
                      <h3>普通嘅h3</h3>

實際效果

上下零距嘅h3

上下零距嘅h3

分得好開嘅h3

分得好開嘅h3

普通嘅h3

普通嘅h3

^以上效果經過加工

用style來改字體

通過style，我哋係可以改變字體嘅大小同粗幼

font-size 係控制字體大少，單位係同margin 一樣郁係用 px

font-weight 係控制字體粗幼，唔洗加單位。正常來講600以上就係粗體

Example Code

                  
                      <h5>普通嘅h5</h5>

                      <h5 style="font-size: 36px">加大版嘅h5</h3>

                      //用分號將特性分開就可以改多過一個嘅style
                      <h5 style="font-size: 36px; font-weight: 900">
                          加大版同變粗嘅h5
                      </h5>

實際效果

普通嘅h5
加大版嘅h5
加大版同變粗嘅嘅h5

^以上效果經過加工

用style來改行高

通過style，我哋係可以改變行高(line-height)

當係有一堆字或者過一行嘅字時就好有用

line-height特性嘅數值可以分為 0（冇行高） 1（唔隔行） 1.5 (隔半行） 同 2 (隔行)

預設嘅line-height係 1 即係唔隔行

Example Code

                  
                      < div style="line-height: 0">
                          呢度係冇行高會貼住下一行
                          <br>
                          呢一行係冇行高會貼住上一行
                      </div>

                      < div style="line-height: 1.5">
                          呢度係會隔半行
                          <br>
                          呢一行同上一行有半行空位
                      </div>

                      < div style="line-height: 2">
                          呢度係會隔一行
                          <br>
                          呢一行同上一行有一行空位
                      </div>

實際效果

呢度係冇行高會貼住下一行
呢一行係冇行高會貼住上一行

呢度係會隔半行
呢一行同上一行有半行空位

呢度係會隔一行
呢一行同上一行有一行空位

^以上效果經過加工
